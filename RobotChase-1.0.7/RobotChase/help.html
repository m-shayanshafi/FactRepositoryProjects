Welcome to Robot Chase: Java Edition

The object of the game is to avoid the robots, pictured above. You can move out of the way using the keyboard or mouse. Try to lure the robots into each other, a post or a bomb. After a crash, the wreckage blocks advancing robots. You can't move into harm's way; jump if you are stuck. You start with a small number of safe jumps; you can get more jumps by luring a robot into a bomb. If you run out of jumps, you may still survive.

Using the keypad: Use the keypad keys 1-9 to move in any of eight directions:


NW 7 North
8		 9 NE
West 4 5 6 East
SW 1 2
South		 3 SE
0 Jump

Pressing the 5 key stays still; the 0 key jumps to a random location. On some keyboards you may need to press the numlock key, or press E to edit the key assignments.

Using the mouse: Click on a nearby square to move in that direction. When animation is on, you keep moving toward that square until you click on a different square or get stuck. Right-click, control-click, or click the high score to jump away.

Keyboard commands:

  • Press A to toggle animation when using the mouse.
  • Press C to change to the next tile theme.
  • Press E to edit the key bindings.
  • Press N to start a new game.
  • Press Q to quit.
  • Press R to reset the high score.

Enjoy!
