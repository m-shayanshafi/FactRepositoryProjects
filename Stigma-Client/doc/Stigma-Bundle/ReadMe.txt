 Requirements
------------------------------

 - Java(TM) SE Runtime Environment version 1.6.0_22 or newer

------------------------------
 Quick Start
------------------------------

Just run the shell script suitable for your operating system:

1. RunMe.bat for Windows
2. run_me.sh for Unix system (bash script)

A script will start a server instance and one client 
which can immediately log in using default setting.

For more detailed information about desired application 
please look into proper ReadMe.txt in that application's directory 
(for example: for more options of server open Stigma-Server/ReadMe.txt).

Enjoy!
