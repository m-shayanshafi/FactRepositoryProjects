------------------------------
 Requirements
------------------------------

 - Java(TM) SE Runtime Environment version 1.6.0_22 or newer

------------------------------
 Running as local application
------------------------------

Just execute provided JAR file. 
By default client tries to connect to server on localhost.

Command line arguments:
 --server-url=  - used to explicitly specify server URL.
 --autologin 	- enables automatic login procedure handling for quick testing. Logs in as the default 'test' user.

To run server - download server package and read ReadMe.txt file there.

------------------------------
 Running as applet
------------------------------

We assume that you want to deploy your own Stigma server:
Download server package and read ReadMe.txt file there.

Enjoy!
